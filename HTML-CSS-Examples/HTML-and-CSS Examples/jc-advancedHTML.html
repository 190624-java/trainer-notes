Headings

There are 6 different levels of headings we can utilize. using <h1> through <h6>.

I'm a second level heading

Notice the "h2" heading is smaller than the "h1", this is because of browser default styling for the heading tags.
The level of the heading should be representative of the importance of the heading. Because you can style an h1 to look the same as an h6, it's important to keep in mind that the different tags represent what kind of content is contained within them.

Lists

There are several different types of lists that you can create in html.

  • The <ul> tag is for unordered lists
  • The <ol> tag is for ordered lists
  • The <dl> tag is for description lists (this is less frequently used)

The "ul" and "ol" tags use the <li> tag for list items.
The "dl" tag uses <dt> and <dd> tags for definition terms and descriptions.

Note: you cannot place a list tag inside of a paragraph tag.

Tables

  • *<caption> - define a caption for the table, if used it must go immediately after the table tag.
  • *<thead> - defines a header row for the table.
  • *<th> - defines a table header cell.
  • *<tbody> - defines the body of the table.
  • *<tfoot> - defines the footer section of a table.
  • <tr> - defines a row of the table.
  • <td> - defines a table data cell in the body.

Forms

Forms are pretty important. You'll use them to get input from the user. A list of common tags are below

  • form - for specifying an entire form.
  • input - represents some kind of user input, depending on the type attribute:
    • text - for inline text
    • number
    • color - displays a color picker
    • date - displays a date picker
    • file
    • range - specifies a range of numbers; use min and max attributes.
    • tel - telephone number
    • password - hides the characters entered.
    • radio - group of options where the user can only select one.
    • checkbox - group of options where the user can select multiple.
  • button - user-clickable button
  • select - creates a drop down menu with options specified by the "option" tag.
  • textarea - multi-lined, resizable text field.
  • label - defines a label for an input element.

HTML5 Semantic Elements

HTML5 is the newest version of HTML, many tags were deprecated but new tags were introduced. Point of new html5 tags is to make HTML markup more semantic - meaning the tags should reflect the meaning of the content, defining more clearly what the content is, not just how it is displayed. Here are some html5 tags:

  • header - defines the heading of a page.
  • footer
  • article - defines an article.
  • aside - content aside from the main content.
  • nav - navigation section.
  • main - main content of the page.
  • section - logically split content into sections.
  • details - additional details a user can view or hide.
  • summary
  • mark - marked/highlighted text
