We can use tags like <div> within the body. A "div" specifies a division of the page. A "div" is a block-level element - which start a new "block" on the next line. As opposed to inline elements - which are created in-line. (span is an example of an inline element.)

The <span> tag is an inline tag- it renders inline with the text around it. Spans are commonly used for styling parts of text inline with other text. the <code> tag is another inline tag.
If we want to make text bold, we can use the "strong" tag or we can use the "b" tag. To italicize, we can use the "i" tag or the "em" tag.

An attribute is a property of a tag that consists of key-value pairs. Attributes go inside of the tag itself in the format of attributeName="attributeValue".
