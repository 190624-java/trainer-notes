CSS - Cascading Style Sheets

Our html has gotten kinda fancy, but it's still ugly and plain. The browser applies some default styling with certain tags, like bolding and heading tags, but there's no fancy colors or fonts or spacing. We use CSS to add styling.

Ways of including styles:

  • inline styling
  • using internal stylesheet(s)
  • using external stylesheet(s)

It is best practice to use external stylesheets., but in this page we will use internal and inline. It's best practice, because it allows for seperation of concerns, basically your html and css will be separated, so that changing a rule in one place will apply to every html page that uses the stylesheet.

Inline Styling

Can be performed on any HTML element using the "style" attribute. Within the quotation marks defining the value of the style attribute, we define a CSS rule which tells the browser what styling to apply to the content of that element. CSS rules are key-value pairs, separated by a colon (:). There are very many, we will not talk about all of them, but here's a simple one.

We will use a span tag to color this word blue. 

 
            <span style="color:blue; background-color:orange"> this text appears blue with an orange background</span>

this text appears blue with an orange background

Internal Stylesheets

In the head tag, we can specify a style tag, within which we can define a CSS stylesheet that will apply to the whole document. A stylesheet is a set of CSS selectors with associated rules for styling our HTML.

A CSS selector is how we select specific HTML elements that we want to style. After the selector we place curly braces - {} - and within the curly braces we write the CSS rules which only apply to elements selected by the CSS selector.

The CSS rules we specified for the body selector, restrict the width of the body to be 800 pixels at maximum and sets the margin to be automatically calculated which will center the element within its parent element.

Selectors

Element Selectors

Use the tag name to identify which elements the CSS rules apply to.


Class and ID selectors

If we want to be more specific about a subset of elements that we want to select, we can use the class and id selectors.
All elements can have the id and class attributes. The id attribute values must be unique on the page. Class attribute values don't have to be unique and are used for the purpose of applying css to all elements with a specified class (or classes).

The class selector is denoted by a period followed by the class name (.classname) and selects all elements whose class attribute's values contain the specified value - which you define.

you can also combine element selectors with class selectors to get more specific. For example, if you want to select all divs with the class "jumbo": 

            div.jumbo { height: 1000px; width: 2000px; }

The id attribute can be applied to any element and uniquely identifies that element. The id selector is denoted by a pound sign, followed by the id (#uniqueIdHere). Say we have an element : <span id="first">I am first!</span> We can target this element as follows: 

            #first {font-size: 32pt;}

Pseudo-class and pseudo-element selectors

A pseudo-class is a specific keyword at the end of a selector used to specify that you want to style the selected element, but only when it is in a certain state - for example, when the user is hovering the mouse over the elements, or when a checkbox is disabled or checked. The syntax is to use a colon (:) to precede the pseudo-class. Here's a short list of some pseudo-class selectors:

  • :active
  • :visited
  • :checked
  • :disabled
  • :first
  • :nth-child()
  • :hover

Pseudo-elements are similar to pseudo-classes, but are prepended by two colons. Some common pseudo-elements are:

  • ::before
  • ::after
  • ::first-line
  • ::first-letter

Child and Descendant Selectors

If you want to be able to only select elements that are nested inside of other elements (for example, selecting all divs within other divs) we use descendant selectors.

There are two ways of selecting nested elements - descendant and direct descendant. To select direct descendants use the > symbol. For example: 

            div > p {...}

If we want to select any descendants of a div tag that are of type p, then we simply separate the selectors by a space. For example: 

            div p {...}

Other Selectors

There are many more other types of selectors, such as sibling selectors, attribute selectors, ... Another helpful selector is selectors that allow you to select an element that has multiple classes or a particular tagname with a particular class (or classes).

If you want to select all elements that have the following classes: class1 class2. Then use the following selector: 

            .class1.class2 {...}

The above selector would select elements that have their class attribute set to "class1 class2", "class1 class2 class3", "class3 class2 class1", etc. as long as both class1 and class2 are in the class values list.

If you want to select all p elements that have class p1: 

            p.p1 {...}

CSS Box Model

The CSS box model is the way that HTML elements are defined - there are actually 4 concentric boxes which are... (from innermost to outermost)

  • Content - this contains the actual text or image
  • Padding - this is the space between the content and the border.
  • Border - space between the padding and margin, you can set its width, style, and color properties.
  • Margin - usually invisible space between the end of one element and the start of another.

Units

There are several different types of values we can provide for styles, each has their own units.

  • Sizes
    • Absolute
      • px - pixels
      • mm - millimeters
      • cm - centimeters
      • in - inches
      • pt - points (1/72 of an inch)
      • pc - picas (12 points)
    • Relative
      • % - precentage relative to parent element's width
      • em - 1em is the same as the font size of the current element
      • rem - stands for "root em" and works the same except refers to the base font size.
      • vh, vw - these are 1/100th of the height and width of the viewport respectively.
  • Colors
    • keywords
    • hex value - # followed by siz hex numbers (0-F)
    • rgb(a) - a function to specify values for red, green, and blue channels.
    • hsl(a) - a function to specify hue, saturation, and lightness values to define a color.
