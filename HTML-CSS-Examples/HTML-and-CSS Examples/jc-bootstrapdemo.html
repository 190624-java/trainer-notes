Hello, Bootstrap!

Welcome to Bootstrap.

This is the easier way to make all of your html pages beautiful and responsive. Bootstrap is a response, mobile-first CSS framework. To fully utilize Bootstrap, you have to use the predefined classes on your html elements.

Two ways to get started with Bootstrap:

  1. Use their CDN (content delivery network) - just include a link tag with the right URL. (which is what we are doing in this page)
  2. Download Bootstrap and link to it locally.

Some useful aspects of Bootstrap:

  • The grid system.
  • responsive by default.
  • components - nicely styled.
  • utility classes - style elements how you want them.
  • consistency - every element will look and feel unified in its design.
